Events App

This is a group project, made by my three classmates and I. I participated on building the front end. This app is made up of Javascript, HTML, SASS, CSS, Mongodb, React and Node J.S. This app is used for creating new events and gatherings. In order to create new events you need to have an account either log in to existing account or create a new one. Once you log in you can create new events by clicking on the NEW EVENT button then fill out the form that is required the title, category, date, time and location. You can also edit or delete the events.
